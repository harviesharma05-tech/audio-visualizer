# 🎵 Audio Visualizer

A minimal audio visualizer with smooth animations. Load your own music and watch it dance!

## ✨ Features

- 📁 **Upload Audio** — Load any MP3, WAV, or OGG file
- 🎨 **4 Styles** — Bars, Circle, Wave, Flower
- 🎨 **Custom Colors** — Pick any color
- ⚡ **Sensitivity Control** — Adjust intensity
- ⏱️ **Timer** — Shows current time / duration
- 💫 **Smooth Animations** — 60fps real-time visualization

## 🚀 How to Use

1. Download `audio-visualizer.html`
2. Double-click to open in browser
3. Click **📁 Load Your Audio**
4. Select your music file
5. Click **▶ PLAY**
6. Watch it visualize! 🎶

## 🛠️ Tech

- HTML5 Canvas
- Web Audio API
- Pure JavaScript (No libraries)
